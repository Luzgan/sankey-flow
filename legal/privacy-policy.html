Sankey Flow — Privacy Policy

Effective date: March 11, 2026

Overview

Sankey Flow is a free Tableau viz extension created by Lukasz Holc. Your privacy is important to us. This policy explains what data the extension accesses and how it is handled.

Data Collection

Sankey Flow does not collect, store, transmit, or share any personal data or workbook data. Specifically:

  • The extension reads summary data from the Tableau worksheet it is placed on using the Tableau Extensions API (getSummaryDataReaderAsync). This data never leaves your browser.
  • No data is sent to any external server, third-party service, or analytics platform.
  • No cookies, local storage, or tracking mechanisms are used to identify or profile users.
  • No personally identifiable information (PII) is collected at any time.

Sandboxed Extension

When installed from the Tableau Exchange, Sankey Flow runs as a sandboxed extension. This means it is hosted by Tableau in a restricted environment that enforces Content Security Policy (CSP) rules, preventing any outbound network requests. The extension cannot communicate with external services even if it attempted to.

Configuration Data

User configuration choices (color palette, layout options, tooltip settings, etc.) are stored within the Tableau workbook using the Extensions API settings mechanism. These settings travel with the workbook and are not sent elsewhere.

Third-Party Libraries

The extension bundles open-source libraries (React, D3.js, d3-sankey, tinycolor2) directly in its JavaScript bundle. No external CDNs or remote resources are loaded at runtime.
